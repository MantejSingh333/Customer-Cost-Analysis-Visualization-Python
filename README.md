# Customer Cost Analysis & Visualization (Python)

## Overview
Cleaned and processed customer product data in Python, then analyzed and visualized cost, revenue, and labor trends over time using Pandas, Matplotlib, and Seaborn to uncover year-over-year patterns.

## Workflow & Tools
- Filtered and cleaned data using **Pandas**  
- Prepared date and text fields for time-series analysis  
- Visualized trends using **Matplotlib** and **Seaborn**  
- Focused on anonymized, structured data to maintain privacy

## Visualizations
- Total Cost Over Time  
- Invoiced Value Over Time  
- Materials Cost Over Time  
- Labor Cost Over Time  
- Labor Hours Over Time  

## Skills
Data cleaning & preprocessing, time-series analysis, Python visualization, translating raw data into actionable insights
